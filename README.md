# DESCRIPTION

ディレクトリ構成はこうすると良さそう、というサンプル。

こうやっても、やっぱりGOPATHはこのツリーのルートに切っておいたほうが良いかも知れない(未検証)。
外部の依存パッケージのソースが近場に集まるので。
